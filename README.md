# SimpleSteps Kids Task Tracker

A simple, parent-friendly kids task tracker with a parent/teacher dashboard, kid login, and task timers.

## What it includes
- Main home page with navigation cards.
- Parent/Teacher dashboard to add kids, create tasks, and assign XP.
- Kids login with PIN.
- Kid dashboard with Start/Pause/Finish/Restart buttons and a timer per task.
- Reports dashboard and app settings.

## Run locally
From the repo root:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000` in your browser.
